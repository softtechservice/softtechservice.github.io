# SoftTech Service Website

A modern, responsive static website for SoftTech Service - an IT company specializing in software development, mobile app development, and customized digital solutions.

## Features

- **Modern Design**: Beautiful, gradient-based design with smooth animations
- **Fully Responsive**: Works perfectly on desktop, tablet, and mobile devices
- **Mobile App Focus**: Highlights Android, iOS, and Flutter development capabilities
- **Service Showcase**: Displays software development, mobile app development, and customized solutions
- **Product Portfolio**: Features HRMS, utility management, and mobile application products
- **Easy to Host**: Ready for GitHub Pages hosting

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- JavaScript (Vanilla JS)
- Font Awesome Icons
- Google Fonts (Poppins)

## Hosting on GitHub Pages

1. Create a new repository on GitHub
2. Upload all files (index.html, styles.css, script.js)
3. Go to repository Settings → Pages
4. Select the main branch as source
5. Your site will be live at `https://yourusername.github.io/repository-name`
