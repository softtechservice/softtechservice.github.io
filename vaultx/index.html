Key Features

Everything you need to take control of your digital security

🔒

Military-Grade Encryption

Your data is encrypted locally using AES-256-GCM encryption. Keys are securely managed by the hardware-backed Android Keystore system — inaccessible to anyone but you.

  • AES-256-GCM encryption standard
  • Hardware-backed Android Keystore
  • Zero-knowledge architecture
  • No master key ever transmitted
👁️

Biometric Unlock

Access your vault instantly and securely using Fingerprint or Face Authentication. Supports strong biometric + device credential fallback.

  • Fingerprint authentication
  • Face authentication support
  • Device credential fallback
  • BiometricPrompt API (OS-level security)
📷

Smart OCR Scanner

Save time with our ML Kit-powered scanner. Automatically extract card numbers, expiry dates, and bank details just by pointing your camera — no manual typing needed.

  • Real-time card data extraction
  • Bank detail recognition
  • Powered by Google ML Kit
  • Images processed locally, never uploaded
📊

Advanced Security Dashboard

Get a bird's-eye view of your security posture. Track your security score, monitor weak passwords, and see upcoming document expiries — all in one glance.

  • Security score based on password strength
  • Category-wise statistics
  • Document expiry alerts
  • Recent & pinned items list
🔑

Password Generator

Create ultra-secure, customizable passwords with the built-in generator. Configure length, character sets, and complexity to meet any requirement.

  • Configurable length & character sets
  • Include uppercase, numbers & symbols
  • One-tap copy to clipboard
  • TOTP (2FA) code support

Clean & Modern UI

A beautiful, intuitive interface built on Material 3 "Glass" design philosophy with full Dark/Light mode support and smooth glassmorphic elements.

  • Material 3 glassmorphic design
  • Full dark & light mode
  • Smooth transitions & animations
  • Favorites & pinned items for quick access

🛡️ Privacy First, Always

VaultX is a local-first application. Your data never leaves your device. We have absolutely no access to your master key or encrypted records. No cloud sync. No analytics. No ads. Just your data, on your device, protected by you.
