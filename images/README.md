# Images Folder

Please add your company logo file here:

- **File name**: `logo.png` (or `logo.svg` if you have a vector version)
- **Recommended size**: 
  - For navigation: 40px height (width will scale automatically)
  - For footer: 35px height
  - Original: At least 200px height for best quality

## Logo Specifications

Based on your logo description:
- Blue gradient square with rounded corners
- White cloud icon with circuit board pathway
- "SoftTech" text in bold
- "SERVICE" text in uppercase

## Supported Formats

- PNG (recommended for transparency)
- SVG (best for scalability)
- JPG (if no transparency needed)

## File Location

Once you add your logo file, it should be accessible at:
- `images/logo.png` (or your chosen filename)

Then update the HTML if you use a different filename or format.
