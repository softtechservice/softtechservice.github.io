📱 Permissions Usage

The game requests minimal permissions, only to provide core functionality:

Permission Purpose
Internet / Network Load advertisements via Google AdMob
Vibration Haptic feedback for taps and wrong moves
Storage (local) Save progress, stats and Speedrun records on device

The game does not request access to your camera, contacts, microphone, or location.

  • All game data is stored locally on your device
