We prioritize user privacy and adhere to data minimization principles. We do not collect, store, or share any personal identity information.

Gameplay Storage

The game stores gameplay progress, achievements, coin count, and diamond count locally on your device via standard browser localStorage. No personal or gameplay database is uploaded to remote servers. This ensures you can play the game offline with full state retention.

Important: Clearing your app storage, cache, or webview data will reset your local gameplay progress, coin balance, and unlock states.

2. Device Permissions

To run the hybrid puzzle canvas securely and offline, the application requests the following device permissions:

Permission Description Purpose
Internet Access `android.permission.INTERNET` / iOS NSNetwork Strictly required to run the local ephemeral asset web server (http://localhost) to render the HTML5 Construct 3 game widget offline. No external server tracking is done.
